### 3.2 Prompt Contract
- 海报必须体现以下机制与视觉要点：
  - 主体是 Hollow Knight 风格的小骑士剪影或近似轮廓，持骨钉站在画面中心或偏下前景。
  - 画面左/右两侧明确区分 `电形态` 与 `冰形态`：
    - 电侧使用金黄闪电、发光电球、快速弧线和高能光效。
    - 冰侧使用冰蓝雾气、冰球、碎晶、盾片或环绕护盾语言。
  - 需要包含 orbiting orb / slot / evocation 的视觉暗示，但不做 UI 截图式排版。
  - 需要保持 HK 氛围：空洞、遗迹、深色背景、强对比发光。
  - 不应加入与当前 Mod 机制不符的第三球种、枪械、科幻机甲或现代 HUD。
- 文案策略：
  - 海报默认不加文字标题，避免与 README 标题重复。
  - 如果模型自动补充少量装饰性字样，可接受，但不主动要求。
