# CodeMap: 白球/冰盾 UI 视觉链路

## 1. Scope
- 目标功能：DeVect 冰盾 UI，也就是白球（`OrbTypeId.White`）的常驻球体视觉与命中特效
- 目标文件：
  - `src/Orbs/Definitions/WhiteOrbDefinition.cs`
  - `src/Visual/OrbVisualService.cs`
  - `src/Orbs/Runtime/OrbRuntime.cs`

## 2. Entry Points
- `DeVect.cs`
  - `HandleShriekCast()` 将上吼路由为白球生成
- `src/Orbs/OrbSystem.cs`
  - `OnShriekCast()` / `HandleSpellCast(OrbTypeId.White)` 进入白球生成链路
- `src/Orbs/Runtime/OrbRuntime.cs`
  - `TrySpawnOrbInSlot(...)` 调用 `OrbVisualService.CreateOrbRenderer(...)` 创建白球渲染器

## 3. Rendering Flow
- `src/Orbs/Definitions/WhiteOrbDefinition.cs`
  - `OrbColor` 定义白球主色，是白球 UI 的基色入口
- `src/Visual/OrbVisualService.cs`
  - `CreateOrbRenderer(...)` 根据 `OrbTypeId.White` 选择 `CreateGlassOrbSprite()`
  - `AddWhiteOrbMaterialLayers(...)` 叠加内发光、切面、高光，决定立体感
  - `SpawnGlassShatterVisual(...)` 生成白球命中的闪光、折射环与碎片，决定冰盾破碎反馈

## 4. Current Visual Reality
- 白球基色当前是偏白的低饱和冷色：`new(0.92f, 0.96f, 1f, 1f)`
- `AddWhiteOrbMaterialLayers(...)` 当前三层都较接近白色：
  - `GlassInnerGlow`: 基色低透明度内光
  - `GlassFacet`: 白色切面
  - `GlassHighlight`: 白色高光
- `CreateGlassOrbSprite()` 的底图亮度分布偏均匀，依赖外部 `renderer.color` 才能拉出风格
- `SpawnGlassShatterVisual(...)` 的环和碎片也是接近白色的冷色，导致命中特效对比度有限

## 5. Risk Boundaries
- 本次只应调整视觉参数与颜色分层，不改白球伤害、触发、索敌、槽位动画
- 不改黑球、黄球视觉语言，避免破坏球种区分
- 颜色要更亮但不能丢失“冰/玻璃”语义，避免变成普通发光白球

## 6. Direct Edit Targets
- `src/Orbs/Definitions/WhiteOrbDefinition.cs`
  - `public Color OrbColor`
- `src/Visual/OrbVisualService.cs`
  - `private static void AddWhiteOrbMaterialLayers(...)`
  - `public void SpawnGlassShatterVisual(...)`
  - 如有必要：`private static Sprite CreateGlassOrbSprite()`
