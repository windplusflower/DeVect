# 骨杖寂骑人

[**English Version**](README-en.md) | **中文说明**

`骨杖寂骑人`，英文名 `DeVect`，是一个把《杀戮尖塔》故障机器人充能球思路移植到《空洞骑士》的 Mod。

## 简介

你不再直接释放三种法术，而是把它们转化为围绕小骑士运转的三类球体：黄球、白球、黑球。它们会在平 A 连段中自动触发被动，也会在球槽满时通过再次施法触发激发效果。

### 名称含义

中文名 `骨杖寂骑人` 对应这次融合的两个来源：

- `骨杖` 取自小骑士的骨钉意象，也呼应把原法术改写为围绕战斗节奏运转的攻击体系
- `寂骑人` 延续了“故障机器人”的谐音趣味，也贴合小骑士沉默、空洞、容器般的身份

英文名 `DeVect` 则是对 `The Defect` 的变形命名。

- 读感上保留了 `Defect` 的机械感与故障感
- `Vect` 可以理解为借自 `Vessel`，对应《空洞骑士》中小骑士作为“容器”的身份
- 合在一起，既像一个失控的机器人名，也像一个脱离既定命运的容器代号

## 核心玩法

- `复仇之魂 / 暗影之魂` 不再直接发射，改为生成 `黄球`
- `嚎叫 / 深渊尖啸` 不再直接释放，改为生成 `白球`
- `下砸 / 黑暗降临` 不再直接释放，改为生成 `黑球`
- 每次施法仍然消耗 Soul，`法术扭曲者` 仍会减少法术消耗
- 你每累计 `3` 次骨钉有效命中，当前所有球都会各自触发一次 `被动`
- 当球槽已满时，再次施放任意对应法术，会把最右边的旧球挤出，并触发那颗旧球的 `激发`

## 球槽规则

| 规则 | 效果 |
|------|------|
| 默认槽位 | 最多同时持有 `3` 颗球 |
| 吸虫之巢 | 球槽扩展为 `4` 格 |
| 新球生成 | 总是从左侧插入，旧球整体向右移动 |
| 激发顺序 | 最右边的球最早离场、最先触发激发 |
| 场景切换 | 保留当前球序和储存数值 |

## 三种球说明

### 黄球 (Lightning Orb)

| 项目 | 效果 |
|------|------|
| 生成方式 | 施放火球法术 |
| 被动 | 对附近随机一个敌人造成约 `1/3` 当前骨钉伤害 |
| 激发 | 对附近随机一个敌人造成 `1x` 当前骨钉伤害 |
| 定位 | 稳定的单体补刀与持续磨血 |

### 白球 (Glass Orb)

| 项目 | 效果 |
|------|------|
| 生成方式 | 施放上吼法术 |
| 初始伤害 | 约 `1/3` 当前骨钉伤害 |
| 被动 | 对范围内所有敌人造成当前储存伤害，然后自身伤害 `-1` |
| 额外规则 | 当伤害降到 `0` 时会直接碎掉并消失 |
| 激发 | 对范围内所有敌人造成 `当前储存伤害 x2` |
| 定位 | 清杂、群伤、滚雪球式范围压制 |

### 黑球 (Dark Orb)

| 项目 | 效果 |
|------|------|
| 生成方式 | 施放下砸法术 |
| 初始储伤 | 约 `0.75x` 当前骨钉伤害 |
| 被动 | 不直接攻击，改为额外储存 `1x` 当前骨钉伤害 |
| 激发 | 攻击范围内当前血量最低的敌人，造成这颗黑球储存的全部伤害 |
| 定位 | 蓄力、定点处决、打精英或 Boss 的高爆发单体球 |

## 护符联动

| 护符 | 效果 |
|------|------|
| `萨满之石` | 不再强化原版法术，改为直接强化三种球的伤害与储伤 |
| `吸虫之巢` | 不再改火球形态，改为让球槽上限从 `3` 变成 `4` |
| `法术扭曲者` | 仍然只影响法术消耗 |
| `灵魂捕手` / `噬魂者` | 当前没有额外改造，保持原本作用 |
| `力量` / `亡者之怒` | 会影响当前骨钉伤害，因此也会间接提高三种球的效果 |

### 萨满之石的具体收益

- 黄球被动和激发直接 `+2` 伤害
- 白球初始伤害 `+2`
- 黑球初始储伤和每次被动储伤都 `+2`

## 玩法理解

- 球的所有伤害都基于你当前的实际骨钉伤害，而不是原版法术伤害
- 黄球会随机锁定附近敌人；白球和黑球使用同一片近身战斗范围
- 黑球的强度很依赖你能否稳定打出 `每 3 次骨钉命中一次被动`
- 白球适合长线压制，黑球适合攒一颗大的再挤出去打爆发，黄球则最稳定

## 推荐玩法

- `黄球流`：持续平 A + 穿插火球，打稳定单体输出
- `白球流`：靠上吼攒白球，利用 3 次命中节奏频繁清场
- `黑球流`：先把黑球养大，再用满槽后的再次施法打定点爆发
- `混合流`：用白球控场、黄球补伤、黑球收头，最像故障机器人的轮转手感

## 提示

- 如果你发现法术没有正常放出来，这是预期行为：它们已经被转成球了
- 想更快触发被动，就尽量保持稳定命中，而不是只靠跑图前攒球
- 装了 `吸虫之巢` 后，球序管理会更重要，因为第 `4` 格会显著改变激发顺序
