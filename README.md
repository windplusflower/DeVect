# 骨杖寂骑人

[**English Version**](README-en.md) | **中文说明**

`骨杖寂骑人`，英文名 `DeVect`，是一个把《杀戮尖塔》故障机器人球系思路移植到《空洞骑士》的 Mod。

本 Mod 围绕 `电形态` 与 `冰形态` 两套技能运转，主要机制包括 `切形态`、`触发被动`、`排队生球`、`满槽激发` 与 `冰盾吸伤`。

## 核心玩法

- `波`：切换 `电形态 / 冰形态`
- `砸`：释放当前形态的小技能
- `吼`：释放当前形态的大技能
- 每次 `切换形态`，都会立刻触发当前场上所有球的 `被动`
- 新球会按顺序排队出现，每颗球之间有短暂间隔
- 球槽满时，新球仍会进入，最右边的旧球会被挤出并触发 `激发`

系统会围绕形态切换、球槽顺序、排队生成与冰盾吸伤持续运转。

## 输入与消耗

这里沿用 `波 / 砸 / 吼` 来称呼三种施法输入。

### 切形态

- 消耗 `1` 次法术费用
- 立即在 `电形态 / 冰形态` 间切换
- 会立刻触发当前所有球的被动

### 小技能

- 消耗 `1` 次法术费用
- `电形态`：生成 `1` 个电球
- `冰形态`：生成 `1` 个冰球

### 大技能

- 消耗 `3` 次法术费用
- `电形态`：生成等于 `当前房间内累计已生成电球数` 的电球
- `冰形态`：生成 `20` 范围内敌人数 `+ 2` 个冰球，并立刻对范围内所有敌人造成 `1x` 当前骨钉伤害；如果装备 `萨满之石`，则改为 `ceil(当前骨钉伤害 x 4/3)`

补充说明：

- `法术扭曲者` 会正常降低单次法术费用，所以也会一起降低切形态、小技能和大技能的总消耗
- 电形态大技能本次额外生成出来的电球，也会计入当前房间累计数
- 如果你在当前房间里还没生成过任何电球，第一次电形态大技能会消耗 Soul，但不会生出新球

## 解锁规则

- `波` 对应 `切形态` 输入
- 只要已经获得 `波系法术`，电形态就能正常生成电球；电球伤害不再区分白波 / 黑波
- `砸` 对应 `小技能` 输入；如果没有获得 `下砸系法术`，冰形态不会生成冰球
- `吼` 对应 `大技能` 输入
- 如果当前形态对应的球种还没解锁，那么该形态的小技能和大技能都不会生成球

## 球槽规则

| 规则 | 效果 |
|------|------|
| 默认容量 | 同时持有 `3` 格球槽 |
| `吸虫之巢` | 球槽上限提升到 `4` 格 |
| 进场顺序 | 新球总是从左侧插入，旧球整体向右移动 |
| 激发顺序 | 最右边的球最早离场，也最先触发激发 |
| 排队生球 | 每颗球依次出现，生成期间你可以移动，也可以继续施法排队，但不能攻击 |
| 场景切换 | 清空当前场上的球、待生成队列，并重置本房间电球累计数 |
| 长椅保存 | 清空当前球和当前冰盾 |

补充说明：

- 形态不会因为切场景而重置
- 冰盾不会因为普通场景切换而清空，但会在长椅保存时清空

## 两种球的设计

## 电球

电球的效果如下。

### 被动

- 对附近随机一名敌人造成 `ceil(当前骨钉伤害 x 1/3)`

### 激发

- 对附近随机一名敌人造成 `ceil(当前骨钉伤害 x 2/3)`

## 冰球

冰球本身不打伤害，只负责积累冰盾。

### 被动

- 获得 `1` 片冰花

### 激发

- 获得 `3` 片冰花

### 冰盾规则

- 每 `4` 片冰花`=` `1` 层可吸伤护盾
- 未满 `4` 片时，只是储存，不会提前挡伤
- 最多累计 `16` 片冰花，也就是最多吸收 `4` 点伤害
- 受到伤害时，会优先按整层护盾吸收

## 数值设计

所有关键数值，都围绕 `当前骨钉伤害` 计算。

- 电球伤害按你当前骨钉伤害结算
- 冰形态大技能的范围伤害默认按你当前骨钉伤害结算；装备 `萨满之石` 时改为 `ceil(当前骨钉伤害 x 4/3)`
- `力量` 与 `亡者之怒` 会先提高你的骨钉伤害，再间接提高这些技能收益
- 为了平衡，小骑士自己的普通骨钉和骨钉技对敌人固定只造成 `1` 点伤害

因此，球伤与冰形态大技能的收益会随着当前骨钉伤害一起变化，而普通骨钉与骨钉技的最终对敌伤害仍固定为 `1`。

## 护符联动

| 护符 | 当前效果 |
|------|----------|
| `萨满之石` | 强化电球的伤害结算，并把冰形态大技能范围伤害提升到 `ceil(当前骨钉伤害 x 4/3)` |
| `吸虫之巢` | 球槽从 `3` 格变成 `4` 格 |
| `法术扭曲者` | 正常降低切形态、小技能、大技能的 Soul 消耗 |
| `灵魂捕手` / `噬魂者` | 保持原本作用 |
| `力量` / `亡者之怒` | 通过提高当前骨钉伤害，间接强化球系收益 |

### 萨满之石

萨满之石有两项效果。

- 电球额外伤害公式：`ceil(当前骨钉伤害 x 0.2 x 本次倍率)`
- 冰形态大技能范围伤害：`ceil(当前骨钉伤害 x 4/3)`

例子：

- 电球被动倍率固定是 `1/3`
- 所以这一下的萨满额外伤害就是 `ceil(当前骨钉伤害 x 0.2 x 1/3)`

补充说明：

- 电球激发同理，额外伤害是 `ceil(当前骨钉伤害 x 0.2 x 2/3)`
- 冰球攒盾本身不受 `萨满之石` 影响
